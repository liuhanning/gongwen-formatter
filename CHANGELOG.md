# 更新日志 (CHANGELOG)

## [v1.1] - 2026-02-08

### ✨ 新增功能

#### 政府交付版格式模式
- 新增政府交付版格式模式，支持标题首行缩进
- 用户可在两种格式模式间自由选择：
  - GB/T 9704标准模式（标题左缩进）
  - 政府交付版模式（标题首行缩进）

#### 命令行增强
- 添加 `--government` 或 `-g` 参数切换到政府交付版模式
- 显示当前使用的格式模式信息
- 改进的帮助提示

#### Web界面改进
- 添加格式模式选择器（单选按钮）
- 现代化的选择器UI设计
- 选中状态高亮显示
- 实时显示使用的格式模式

#### API增强
- `GongwenFormatter` 类新增 `format_mode` 参数
- 新增格式模式常量：`FORMAT_MODE_STANDARD` 和 `FORMAT_MODE_GOVERNMENT`
- `create_demo_document()` 和 `format_existing_document()` 支持模式参数

### 📝 文档更新
- 更新 README.md，添加格式模式说明
- 新增《格式模式说明.md》详细对比文档
- 新增《政府交付版实施总结.md》技术文档
- 新增《快速开始-政府交付版.md》快速指南

### 🔧 技术改进
- 重构样式配置，分离两种模式的样式定义
- 优化代码结构，提高可维护性
- 改进错误处理和用户提示

### 📊 文件变更
- ✏️ 修改：`gongwen_formatter.py` - 核心引擎支持双模式
- ✏️ 修改：`web_server.py` - API支持模式参数

## [v1.0] - 2026-02-04

### ✨ 初始版本

#### 核心功能
- 完整的公文格式化引擎
- 符合 GB/T 9704-2012 标准
- 7级标题样式支持
- 自动标题级别识别
- 表格和插图格式化
- 页码和页眉设置

#### 多种使用方式
- Web界面（Flask + 现代化前端）
- Python命令行工具
- Python API
- VBA宏（Word和WPS）

#### 字体兼容性
- 自动检测系统字体
- 字体退化方案
- 中英文字体分离处理

#### 文档
- 完整的 README.md
- WPS兼容性说明
- 使用示例和API文档

#### 支持的格式
- 页面设置（A4、边距、页眉页脚）
- 标题格式（一级到六级）
- 正文格式（仿宋、行距、缩进）
- 表格格式（标题、表头、内容）
- 插图格式（标题、位置）
- 序号规则（一、（一）1．（1）①）
- 符号校正（中英文标点）
