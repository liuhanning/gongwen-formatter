# 🚀 使用指南 - 政府交付版格式模式

## 快速选择

**问：我应该用哪种模式？**

| 你的文档类型 | 推荐模式 |
|------------|---------|
| 党政机关公文、正式行政文件 | GB/T 9704标准 |
| 政府交付报告、咨询研究报告 | 政府交付版 ✨ |

---

## 三种使用方式

### 🌐 方式一：Web界面（推荐新手）

```bash
# 1. 启动服务器
python web_server.py

# 2. 打开浏览器访问
http://localhost:5000

# 3. 在界面上选择格式模式
⭕ GB/T 9704标准（标题左缩进2字符）
⭕ 政府交付版（标题首行缩进2字符）✨

# 4. 上传文档，点击"格式化文档"
```

### 💻 方式二：Python命令行（推荐熟练用户）

```bash
# 标准模式（默认）
python gongwen_formatter.py input.docx output.docx

# 政府交付版模式 ✨
python gongwen_formatter.py input.docx output.docx --government
# 或使用简写
python gongwen_formatter.py input.docx -g

# 生成示例文档
python gongwen_formatter.py              # 标准模式
python gongwen_formatter.py --government # 政府交付版
```

### 📄 方式三：VBA宏（Word/WPS内使用）

```
1. 导入宏文件
   - Word用户：GongwenFormatter.bas
   - WPS用户：GongwenFormatter_WPS.bas

2. 运行宏
   - 按 Alt + F8
   - 选择 "FormatGongwen" 或 "格式化公文"
   - 点击"运行"

3. 选择格式模式
   ┌─────────────────────────────┐
   │   请选择格式模式：           │
   │                              │
   │ 【是】= GB/T 9704标准模式    │
   │       （标题左缩进2字符）     │
   │                              │
   │ 【否】= 政府交付版模式       │
   │       （标题首行缩进2字符）   │
   │                              │
   │ 【取消】= 取消操作           │
   └─────────────────────────────┘

4. 等待格式化完成
```

## 两种模式的区别

### 视觉对比

#### GB/T 9704标准模式
```
        一级标题（居中）

  一、二级标题
  正文内容首行缩进...

  （一）三级标题
  正文内容首行缩进...
```

#### 政府交付版模式 ✨
```
        一级标题（居中）

一、二级标题
  正文内容首行缩进...

（一）三级标题
  正文内容首行缩进...
```

### 核心差异

| 标题级别 | GB/T 9704标准 | 政府交付版 |
|---------|--------------|-----------|
| 二级标题 | 左缩进2字符 | 首行缩进2字符 |
| 三级标题 | 左缩进2字符 | 首行缩进2字符 |
| 四级标题 | 左缩进2字符 | 首行缩进2字符 |

## 常见问题

### Q1: 如何切换模式？
**A:** 重新运行格式化工具并选择不同的模式即可。

### Q2: 两种模式可以混用吗？
**A:** 不建议。一个文档应该统一使用一种模式。

### Q3: 如何判断文档当前是哪种模式？
**A:** 查看二级标题：
- 整个标题左移 → GB/T 9704标准
- 只有首行缩进 → 政府交付版

### Q4: VBA宏在WPS中能用吗？
**A:** 能！请使用 `GongwenFormatter_WPS.bas` 以获得最佳兼容性。
